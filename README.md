# Examwork
一个Markdown编辑器"Markedit"项目。

## 功能介绍

Markedit为一个轻量级的、简易的Markdown编辑器。  

拥有作为一个文本编辑器基本的功能，包括对文件的**增、删、查、改**；但与此同时，作为一个markdown编辑器，Markedit支持大部分md官方的语法，支持**markdown实时预览**和**包括html标签和html字符实体的语法高亮**，基本涵盖了所有markdown编辑的通用功能。

此外，在原生markdown编辑器的基础上，Markedit还增加了 **“笔记”** 功能。特别是需要用markdown记笔记、写大纲的情况下，能方便地整合各个markdown文件，形成网络结构。

## 安装

下载发行的压缩包，解压运行Markedit.exe即可使用。（目前只有Windows 64-bit的发行版）
